# Anno Achmat Istiana - Personal Portfolio

link website on vercel: https://annoportofolio.vercel.app/

A modern, minimalist dark-mode personal portfolio website built with Next.js (App Router), Tailwind CSS, Framer Motion, and Lucide React.

## Features

- **Modern Minimalist Design:** Clean, dark-themed, and professional look.
- **Fluid Animations:** Integrated with `framer-motion` for smooth cursor following, scroll reveals, and section transitions.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop viewing.
- **Core Sections:** Hero, About Me, Skills, Projects, and Contact.
- **Performance Optimized:** Built on the Next.js App Router for optimal rendering and SEO.

## Tech Stack

- **Framework:** Next.js 16 (React 19)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/`: Contains the main layout, routing, and globals CSS.
- `src/components/`: Reusable modular UI components (`HeroSection`, `AboutSection`, `ProjectsSection`, etc.).
